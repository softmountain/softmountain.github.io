（图论初学）单源最短路径：dijkstra

目标：寻找起始点到目标点的最短路径，做法，每次扫描最短出边，加入路径。。。

流程如下：1.初始化d[1]=0,然后将其余的dist全部设置为最大
2.进行n-1次循环，每次都找最小了dist[x],将x标记
3.扫描该x的全部出边（x,y,z）,if(d[y]>d[x]+z),更新d[y]

#include<algorithm> #include<iostream> #include<string> #include<cstdlib> #include<cstring> using namespace std; const int maxn=105; const int INF=0x3f3f3f3f; int d[maxn],a[maxn][maxn],v[maxn],ans,m; void search(){ memset(v,0,sizeof v); for(int i=1;i<m;i++){ int x=0; for(int j=1;j<=m;j++){ if(!v[j]&&(x==0||d[j]<d[x])) x=j; } v[x]=1; for(int j=1;j<=m;j++){ d[j]=min(d[j],d[x]+a[x][j]); } } } int main(){ cin>>m; string s; for(int i=1;i<=m;i++){ a[i][i]=0; } for(int i=2;i<=m;i++){ for(int j=1;j<i;j++){ cin>>s; if(s=="x"){ a[i][j]=a[j][i]=INF; } else{ int n=0; for(int k=0;k<s.size();k++){ n=n*10+s[k]-'0'; } a[i][j]=a[j][i]=n; } } } for(int i=2;i<=m;i++){ d[i]=INF; } d[1]=0; search(); ans=0; for(int i=1;i<=m;i++){ ans=max(d[i],ans); } cout<<ans<<endl; }

时间复杂度O(n^2)…适用于求单源路径…
